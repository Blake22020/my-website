M4 CSL

3.0-литровый битурбированный S58 с 550 лошадиными силами — самый мощный серийный двигатель в истории BMW M. Разгон до 100 км/ч за 3.5 секунды — на грани физики для заднеприводного купе. Максимальные 307 км/ч — скорость, где аэродинамика становится религией. Полностью карбоновый капот, титановые пружины подвески и отсутствие задних сидений — каждый грамм лишнего веса принесен в жертву скорости. CSL — для тех, кто верит: настоящий спорткар рождается только когда инженеры сходят с ума.
